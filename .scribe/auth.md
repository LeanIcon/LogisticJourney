# Authenticating requests

To authenticate requests, include a **`Authorization`** header with the value **`"Bearer {YOUR_AUTH_TOKEN}"`**.

All authenticated endpoints are marked with a `requires authentication` badge in the documentation below.

Authenticate using a Bearer token from Sanctum. Example: <code>Bearer 14|KQEecOsc6NCmfB47ESS5wqJmB0KyydIHx9QcHnkP6c319277</code>.
